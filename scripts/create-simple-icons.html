This tool creates simple PWA icons for your app. Click the button below to generate and download all required icon sizes.

Manual Instructions

  1. Open this HTML file in a browser
  2. Click "Generate & Download All Icons"
  3. Move all downloaded PNG files to champbase/public/
  4. Rebuild your app: npm run build
  5. Test installation on mobile device

Alternative: Use Your Own Logo

For a professional look, use the scripts/generate-pwa-icons.js script with your custom logo:

  1. Install sharp: npm install --save-dev sharp
  2. Add your 1024x1024 logo as public/icon-source.png
  3. Run: node scripts/generate-pwa-icons.js
